# buzzwords-frontend

## 项目介绍

* 前端代码，现已经编译为微信小程序和H5
* 使用 [vue-cli](https://uniapp.dcloud.io/quickstart?id=%e5%88%9b%e5%bb%bauni-app-1) 构建项目模版
* 基于 uni-app, 使用 [color-ui](https://www.color-ui.com/) 组件库

## Project setup
```
npm install
```
